DeviceInputForwarderSample
==========================

DeviceInputForwarderSample

スマートフォン等の入力をUDPで指定したIPへ転送するアプリケーションのサンプルです。
サンプルの動作にはStandard AssetsのCharacter Controllersが必要です。
また、使用するUDPポートの解放が必要になることがあります。

クライアントは此方から取得してください。
